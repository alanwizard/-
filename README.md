Ответ на бизнес вопрос: Наибольшое влияние на время доставки оказывают загруженность трафика, наличие фестиваля и рейтинг курьера. Умеренное влияние оказывают такие факторы как: состояние транспорта, тип транспорта и погодные условия.
В свою очередь возраст курьера и тип заказа практически не имеют никакой корреляции с временем доставки заказа.
